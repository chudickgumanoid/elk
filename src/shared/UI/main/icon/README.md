# m-icon Component

Компонент для иконок, создаются свг спрайты

## Использование

```vue
<template>
  <!-- будет показана иконка поиска с нужным цветом -->
  <m-icon
    name="search"
    class="!text-[#79747E]"
  />
</template>
```

## Компоненты

### MIcon

#### Props

- `name`: String (required: true) - Имя иконки, обязательный пропс
- `size`: String | Number (default: '24px') - размер иконки, по умолчанию 24px

#### Дополнительно

При необходимости перекрасить иконку, можно использовать класс tailwind, для изменения свойства color. Пример: `!text-[#79747E]`

### ВАЖНО

Новые добавленые иконки должны иметь `fill|stroke` в значении `currentColor`, иначе перекрасить иконку будет невозможно
